### 설명
> * 카카오 API를 이용하여 지도를 띄우고 메인페이지를 표출한다. 
> * 사용자가 구를 검색해 동을 선택하면 인구수를 보여주고 유동인구, 경쟁업체수, 총인구, 구매력, 매출액대비 영업이익 등
5개의 변수로 생성된 방사형그래프를 지도의 오른쪽에 띄움으로써 상권분석의 결과를 보여준다.
> * 상권분석의 척도는 좋음, 양호, 고려, 나쁨으로 구분해 사용자가 시각적으로 판단할 수 있게 해 준다.
